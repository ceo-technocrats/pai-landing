사용자

파이는 누구를 위한 AI인가요?

초선부터 재선까지, 의원실 모두를 위해 설계했습니다.

초선 의원
3년 걸리는 업무 파악,
오늘부터 시작하세요.
  • 무엇부터 해야 할지, 어디서 찾아야 할지 막막함
  • 우리 지역 조례가 몇 개인지, 타 지역과 뭐가 다른지 모름
  • 첫 발언문·질의서를 혼자 써야 하는 두려움
  • 지역 현안 따라잡는 데만 몇 달
파이와 함께라면
  • 우리 의회 조례 현황 즉시 파악
  • 지역 이슈 브리핑으로 임기 첫날부터 현장 감각
  • 발언문·질의서 초안, 데이터 기반으로 자동 생성
  • 타 지자체 사례 비교로 검증된 논리 구축
재선 의원
경험은 충분합니다.
이제 속도를 올리세요.
  • 행감 질의서 준비에 매번 몇 주 소요
  • 조례 발의 위해 타 지역 비교·초안 작성에 몇 달
  • 예산 심의 자료 검토에 반나절씩 투자
  • 아는데 시간이 없어 준비가 얕아지는 악순환
파이와 함께라면
  • 행감 질의서: 몇 주 → 몇 시간
  • 조례 발의: 전국 비교 + 초안, 당일 완성
  • 예산 심의: 핵심 쟁점 자동 요약 + 근거 출처
  • 더 깊이 파고드는 전문성으로 차별화
정책지원관
더 깊은 리서치,
더 빠른 보고.
  • 5~6개 사이트 교차검색에 반나절 소진
  • 수백 건 검색결과 수동 필터링, 맥락 파악 불가
  • 시간에 쫓겨 자료 완성도가 떨어지는 한계
  • 의원 요청에 즉시 대응하기 어려운 구조
파이와 함께라면
  • 조례·회의록·법령 통합검색, 즉시 결과
  • 의원별·시기별 발언 자동 요약 + 원문 링크
  • 더 완성도 높은 자료로 의원 신뢰 확보
  • 반복 리서치 대신 분석·기획에 집중

왜 파이인가

범용 AI는 그럴듯한 답,
파이는 정확한 답 + 출처

"그냥 GPT 쓰면 안 돼?" — 의정활동에서는 안 됩니다.

기능 ChatGPT 등 범용 AI 파이

의회 속기록 ✗ 접근 불가 ✓ 의미 기반 검색

개인정보·보안 ⚠ 입력 데이터 학습 ✓ 격리 환경, 학습 없음
