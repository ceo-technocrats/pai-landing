좋은 AI 에이전트는 묻기 전에 먼저 챙깁니다.

⚖️
상위법 개정 시
법이 바뀌었습니다
담당 분야 상위법이 개정되면 즉시 알림. 조례 제정·개정 기회를 선제적으로 파악합니다.
🏆
타 지자체 제정 시
전국 최초 조례입니다
타 지자체에서 처음 제정된 조례를 즉시 감지. 우리 의회가 앞서갈 기회를 놓치지 않습니다.
📅
회의 전날 자동
내일 회의 브리핑입니다
다가오는 회의 안건을 자동 분석하고 브리핑 문서를 생성. 의원이 회의에 최적으로 준비됩니다.
📡
관심 분야 모니터링
이 이슈, 주목하세요
설정한 관심 분야의 뉴스·정책 동향을 자동 수집. 지역 현안을 놓치지 않고 선제 대응합니다.

조례 초안
조례 발의
표준 형식 + 전국 유사 조례 참고 + 상위법 근거 포함. 중복 조례 자동 검토.
📊
상임위 설명 자료
심사 대비
필요성 근거 + 타 지자체 사례 + 예상 질의 대응 자료까지 한 번에.
🎤
5분 발언문
본회의
이슈 배경 + 통계 근거 + 타 지자체 사례 + 촉구 사항, 5분 분량으로 자동 구성.
📰
보도자료
언론 배포
핵심 메시지 + 배경 설명 + 인용문 형식. 바로 배포 가능한 수준으로 작성.
🔎
행감 질의서
행정감사
예산 분석 + 과거 발언 기반 질문 + 원문 출처 포함. 근거 있는 질의를 즉시.
📋
공약 리포트
대외 소통
공약별 이행 현황 자동 정리. 유권자 소통과 의정 성과 보고에 바로 활용.
6종의 문서, 모두 데이터 기반 + 출처 포함. 초안이 나오면 의원은 판단과 수정에만 집중합니다.
